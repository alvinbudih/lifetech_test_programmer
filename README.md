## Run Locally

Go to the server-side directory

```bash
  cd server-side/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Go to the client-side directory

```bash
  cd client-side/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
